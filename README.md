# 🚀 FIBPlan

Una eina intel·ligent per als estudiants de la **Facultat d'Informàtica de Barcelona (FIB)**. Cansat de passar hores combinant grups de teoria, problemes i laboratoris? Deixa que l'algorisme trobi el teu horari ideal.

## 🎯 Què fa aquesta app?
Aquesta aplicació web consumeix les dades oficials de l'**API de la FIB (v2)** per generar combinacions d'horaris possibles i permetre't filtrar-les segons les teves condicions personals.

### Funcionalitats clau:
- **Optimització d'horaris:** Defineix les teves preferències (franja horària, dies lliures, màxim d'hores seguides).
- **Cerca intel·ligent:** Evita solapaments de manera automàtica.
- **Filtres personalitzats:** Prioritza determinats professors o tipus de grups.
- **Integració oficial:** Dades actualitzades directament de l'API de la FIB.

## 🛠 Stack Tecnològic
*(Personalitza aquesta secció segons el que utilitzis):*
- **Frontend:** React / Next.js
- **Backend:** Node.js / Python (FastAPI)
- **API:** [FIB API v2](https://api.fib.upc.edu/v2/docs/)

## 🏗 Com començar (Desenvolupament)

1. **Clona el repositori:**
   ```bash
   git clone https://github.com/DjibrilBiel/FIBPlan.git
   ```

2. **Instal·la les dependències:**
   ```bash
   # Si utilitzes Node.js:
   npm install

   # O si utilitzes Python:
   pip install -r requirements.txt
   ```

3. **Configura les variables d'entorn:**
   Crea un fitxer `.env` seguint l'exemple del fitxer `.env.example`.

4. **Executa el projecte:**
   ```bash
   npm run dev
   ```
